Lesson 2

Professions and Careers in Computing

In the contemporary world, computers are integral to various facets of life, with constant advancements in applications. Their widespread use extends across academia, research, industry, government, and businesses. As technology becomes even more ubiquitous, the potential for diverse careers in computer-related fields expands. Graduates in these fields can explore opportunities in programming, software development, information systems, telecommunications, computer science research, graphics, multimedia, web development, internet-related roles, as well as training and support.

Lesson 3

Information Processing Cycle

The Information Processing Cycle involves input, processing, storage, and output, with data being raw facts and information as processed, organized data. The four components of a computer system are hardware, software, people ware, and data ware. Hardware includes input/output devices and storage. Software comprises system and application software. People ware refers to users, and data ware involves data management tools. The text emphasizes the pandemic's impact on purchasing power. Examples illustrate data vs. information and hardware components. Overall, the content introduces key concepts in computing.

Lesson 5

HTML and CSS

HTML, or Hyper Text Markup Language, is explained as the standard markup language for web pages, utilizing tags to structure content. CSS, or Cascading Style Sheet, is a design language that controls the visual presentation of HTML elements. The chapter covers key HTML tags, CSS selectors for element styling, and the distinctions between CSS and JavaScript. Additionally, it introduces wireframes as low-fidelity representations for early UI structure decisions and mockups as higher-fidelity depictions, resembling the final product, useful for client presentations and design approval.
